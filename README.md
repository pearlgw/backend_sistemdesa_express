# Setup Project

Berikut adalah langkah-langkah untuk menyiapkan dan menjalankan proyek ini.

## 1. Install npm
```bash
npm install
```

## 2. Install Nodemon secara Global (opsional saja)
```bash
npm install -g nodemon
```

## 3. Salin File Konfigurasi .env
```bash
cp .env_example .env
```

## 4. Setup Database
*config/Database.js*
